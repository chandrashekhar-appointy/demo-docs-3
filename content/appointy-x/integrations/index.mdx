#### [Zoom](/appointy-x/integrations/zoom)

Integrate Zoom meetings with Appointy X for virtual appointments and services. This integration enables automatic meeting creation and management for virtual sessions.

Key features:

- Automatic Zoom meeting creation for virtual appointments
- OAuth-based authentication for staff members
- Secure meeting management and configuration
- Comprehensive meeting settings control
